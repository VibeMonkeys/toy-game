# 🎮 최진안의 이세계 모험기 - 완전 통합 기획서

## 📋 프로젝트 정의
- **장르**: RPG × 로그라이크 하이브리드 (Hades 스타일)
- **핵심 컨셉**: 퀘스트 중심 스토리 + 반복 플레이 메타 프로그레션
- **플랫폼**: 웹 브라우저 (HTML5 Canvas)
- **목표 플레이타임**: 3-5시간 (첫 완주), 무한 반복 가능

---

## 🎯 게임 핵심 루프

```
[ 시작 ]
    ↓
[ 영혼 성소 ] ← 여기서 업그레이드, 무기 선택, 퀘스트 확인
    ↓
[ 던전 진입 (1층) ]
    ↓
[ 적 처치 + 퀘스트 진행 ] ← RPG 요소
    ↓
[ 아이템 수집 + 파워업 ] ← 로그라이크 요소
    ↓
[ 보스 처치 → 다음 층 ]
    ↓
[ 사망 시 → 영혼 성소로 귀환 ] ← 소울 포인트 획득
    ↓
[ 영구 업그레이드 + 새 무기 해금 ]
    ↓
[ 다시 도전 (더 강해짐) ]
```
